## Polaris
This project was made for the course Individual Propedeuse Assessment. It is a Proof of Concept.

The used technologies for the backend are:
  * PostgreSQL database
  * REST
  * DAO design pattern 

For the frontend:
  * HTML
  * CSS (Bootstrap as framework)
  * JS
 
The used PaaS is Heroku.
